Detailed Description

This class implements the adapted turnip method with links to estimate the unreliability of a network.

The discrete probabilities are computed at the beginning before any run is launched. We sample by rejection; when a link s has been sampled, we remove it from further sampling. In this class, we do not generate the link repair times, only the permutation.
