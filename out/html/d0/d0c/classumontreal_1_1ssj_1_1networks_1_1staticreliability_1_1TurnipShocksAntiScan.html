Detailed Description

This class implements the turnip algorithm with reverse scan to estimate the unreliability of a network.

We generate all shock times and then sort them in time order. It uses a reverse scan of the shocks similar to a constructive schema where all the shocks have occurred initially. Then the shocks are removed one by one until the network becomes operational.
