Detailed Description

Implements the generalized splitting algorithm.

This algorithm is used to obtain an estimate of a rare event probability via Gibbs sampling. See Zdravko thesis for more details. The main parameters to run the algorithm are:

  • a Markov chain
  • a number of Markov chain at the beginning of the simulation
  • the splitting factors
  • the gamma levels
