Detailed Description

This class implements the Permutation Monte Carlo method with shocks to estimate the unreliability of a network.

The discrete probabilities are computed at the beginning before any run is launched. We sample by rejection, generating only shocks as needed; when a shock has been sampled, we remove it from further sampling. In this class, we do not generate the shock times, only the permutation.
