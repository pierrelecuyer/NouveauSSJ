Detailed Description

This class implements a stochastic non-oriented weighted graph (a network).


Non oriented weighted graph
A Graph object is defined by a set of Node's and a set of Link's. We assume that each node is connected with at least one another node, and a link connect only two nodes. Hence, it is a non-oriented graph which connects the source and the target.

Construction of a Graph
We assume a Graph is built from a text file which must be as follows:

NumberOfNodes NumberOfLinks SourceNode TargetNode
NodeI NodeJ Reliability
...
We give an example of a basic network connecting corners of a square
4 4 0 2
0 1 0.9
0 3 0.8
1 2 0.6
2 3 0.8

The choice to give the reliability inside the text file is arbitrary. If we assume Graph is homogeneous (i.e. each link has the same reliability), we can change the reliability of the network with one of its method. An update should give the possibility of relaxing the homogeneous assumption.
