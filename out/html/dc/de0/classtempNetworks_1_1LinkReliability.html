◆ calcParam()

void calcParam ( SamplerType  sam)
private

Given the reliability r of the link, computes and sets the parameter of the distribution to generate random variables, and set the param in this object.

So that we don't have to compute the same log or inverse normal repeatedly for each random number generated. This is done for speed.

  • For uniform sampling, param = 1/r is the upper limit of the range U[0, param].
  • For exponential sampling, param = -log(1-r) is the rate of the exponential distribution.
  • For normal sampling, param = 1/NormalDist.inverseF01(r) is the standard deviation of the standard normal distribution.

samwhich kind of random sampler is used
