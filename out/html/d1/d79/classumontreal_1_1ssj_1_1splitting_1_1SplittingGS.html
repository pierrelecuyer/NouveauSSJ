Bin counters for the log10 of the estimate for each run.

There are s bins dividing the interval [a, b] equally. If flag is true, use bin counters, otherwise not. By default, the bin counters are not used.

Store the estimate W for each run in a TallyStore.

The TallyStore has a maximum capacity of n values. Sets flag true to store each value, false otherwise. Usually, values for each run are not stored and only the average is kept.

Parameters
flagif true, store the values, otherwise not
nnumber of values to store (number of runs)
